For fun\
Using octile distance as a heuristic

### Keybinds
Left Click to place start and end tile\
Hold F to place obstacles\
After Start and End is place press spacebar to run the algorithm\
Once the algorithm replay animation is done press space bar to reset

### What each of the coloured tiles means
**Start:** Dark Green\
**End:** Dark Blue\
**Obstacle:** Black\
**Path:** Yellow\
**Frontier/Open:** Green\
**Closed:** Skyblue\
**Unvisted:** Gray
